IR Sensor System
IR Beam Simulation
FC-51 INFRARED OBSTACLE SENSOR · 3.3V–5V · ESP32 GPIO
⚡

Instant Detection

The FC-51 IR sensor pair emits a 38kHz infrared beam. When a vehicle interrupts the beam, the receiver signals HIGH to the ESP32 GPIO pin in under 50ms.

🔗

ESP32 Integration

Each sensor is wired to a dedicated GPIO pin on the ESP32. The firmware polls all pins at 20Hz and publishes state changes via MQTT to the backend.

🌐

Live Web Sync

MQTT messages received by the Node.js server push WebSocket events to all connected web clients instantly.

🔋

Low Power Design

Each IR sensor consumes only ~20mA at 5V. The ESP32 uses deep-sleep between polling cycles for solar-powered deployments.
