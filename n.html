IR Sensor System

FC-51 IR sensors detect vehicles in under 50ms. ESP32 publishes state via MQTT → Node.js updates MongoDB Atlas → frontend refreshes slot grid.

MongoDB Atlas Storage
Every slot state, booking, OTP, and session is stored in MongoDB Atlas. The 6-digit OTP is saved to the bookings collection and verified at entry/exit.
🔐
OTP Verification Flow
After payment, a 6-digit OTP is generated and stored in MongoDB. The barrier calls POST /verify-otp to look it up and grant entry.
🚗
Exit & Session History
Exit validates the OTP, calculates actual duration and fee, frees the slot, and writes an immutable record to the sessions collection.
